# Hướng Dẫn Sắp Xếp File  

Đối với các bài có file `.txt`, các bạn mở riêng từng thư mục để tránh lỗi. Hoặc có thể di chuyển các file `.txt` ra ngoài theo định dạng sau:  

TongHop/
    ├── DeQuy/
    │   ├── Bai1.py
    │   ├── Bai2.py
    ├── Ham_Def/
    │   ├── Bai1.py
    │   ├── Bai2.py
    ├── ...
    ├── File .txt
